# dashboard/ ফোল্ডার গাইড

## উদ্দেশ্য

এই ফোল্ডারটি SupremeAI-র মূল রিয়াক্ট ড্যাশবোর্ড অ্যাপ্লিকেশন ধারণ করে। এটি এডমিন প্যানেল এবং ইউজার ইন্টারফেসকে কভার করে।

## ফোল্ডার স্ট্রাকচার

```
dashboard/
├── src/           # রিয়াক্ট সোর্স কোড
├── dist/          # বিল্ড আউটপুট
├── node_modules/  # নোড মডিউল
├── package.json   # নোড প্যাকেজ কনফিগ
└── vite.config.ts # ভিট বিল্ড কনফিগ
```

## প্রয়োজনীয় কমান্ড

```bash
# ডেভেলপমেন্ট সার্ভার চালান
npm run dev

# প্রোডাকশন বিল্ড
npm run build

# টেস্ট রান করন
npm run test
```

## রুট রাউটিং

- `/` - ড্যাশবোর্ড হোম
- `/admin/*` - এডমিন প্যানেল (সম্পূর্ণ একটি ইউআই)
- `/admin/dashboard` - এডমিন ড্যাশবোর্ড
- `/admin/security` - সিকিউরিটি ম্যানেজমেন্ট
- `/admin/learning` - লার্নিং সিস্টেম

## গুরুত্বপূর্ণ কম্পোনেন্ট

- `AdminRouteLayout.tsx` - এডমিন রাউট লেআউট
- `App.tsx` - মূল অ্যাপ কম্পোনেন্ট
- `DashboardConfigs.tsx` - ড্যাশবোর্ড কনফিগারেশন
