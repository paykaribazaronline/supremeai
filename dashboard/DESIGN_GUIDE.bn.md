# 🥇 সুপ্রিম AI সিনেমাটিক ডিজাইন গাইড

এই ডকুমেন্টটি **ড্যাশবোর্ড হোম**কে সুপ্রিম AI প্রকল্পের মধ্যে সকল UI/UX ডেভেলপমেন্টের বেসরেখা হিসেবে একাড করে। ভবিষ্যৎ সকল ট্যাব এবং কম্পোনেন্টের জন্য "মডার্ন অ্যাডমিন" স্ট্যান্ডার্ডস মেনে চালাতে হবে যাতে একটি স্বজনীয়, হাই-টেক, সিনেমাটিক অভিজ্ঞতা নিশ্চিত হয়।

---

## 🎨 ১. কোর ভিজ্যুয়াল আইডেন্টিটি

### কolor প্যালেট (নিয়ন ও অবসিডিয়ন)
শুধুমাত্র `index.css`-তে ডিফাইন করা CSS ভেরিয়েবলগুলো ব্যবহার করুন:
- **প্রাইমারি অ্যাকসেন্ট**: `--neon-blue` (`#00f3ff`) - "সিস্টেম অনলাইন", প্রাইমারি অ্যাকশন এবং স্থিতিশীল স্ট্যাটাসে ব্যবহৃত হবে।
- **সেকেন্ডারি অ্যাকসেন্ট**: `--neon-purple` (`#bc13fe`) - ইন্টেলিজেন্স মেট্রিক্স, এমএল টাস্ক এবং হাই-টেক ফ্ল্যারে ব্যবহৃত হবে।
- **সাক্সেস**: `--success` (`#10b981`) - "হেলথি" বা "অপারেশনাল" স্টেটে ব্যবহৃত হবে।
- **ডেঞ্জার**: `#ff4d4f` (নিয়ন লাল) - বিকল্পবাদ বা "ফেইলড" স্টেটে ব্যবহৃত হবে।
- **বেস**: `--cyber-black` (`#020205`) সাথে গ্লাস ওভারলে যুক্ত করুন।

### টাইপোগ্রাফি
- **শিরোনাম**: `'Outfit', sans-serif` (বল্ড/এক্সট্রাবল্ড, লেটার স্পেসিং: -1px)।
- **টেকনিকেল/ডাটা**: `'JetBrains Mono', monospace` (টাইমস্ট্যাম্প, আইডি এবং লগের জন্য ব্যবহৃত হবে)।

## 🧱 ২. UI কম্পোনেন্টস (গ্লাস স্ট্যান্ডার্ড)

### গ্লাস কার্ড (`.glass-card`)
প্রতিটি মেজর সেকশনকে একটি গ্লাস কার্ডে লাপ টানতে হবে।
```tsx
<div className="glass-card" style={{ height: '400px' }}>
  <div className="glass-card-title">
    সেকশন শিরোনাম
    <IconComponent style={{ color: 'var(--neon-blue)' }} />
  </div>
  {/* কন্টেন্ট */}
</div>
```
*নিয়ম:* 
- ব্যাকড্রফিট: `blur(12px)`।
- বর্ডার: `1px solid rgba(0, 243, 255, 0.15)`।
- প্যাডিং: `24px`।

### বাটনস
- **প্রাইমারি**: `.cyber-button` (হোভারে গ্লো)।
- **সেকেন্ডারি/আইকন**: `.glass-action-button` (গস্ট বর্ডার, সাবলে ব্লার)।
- **ডেঞ্জার**: `.cyber-danger-button` (ক্রিমসন গ্লো)।

### গ্রেডিয়েন্টস
এমনাম সম্মর্থিত শিরোনামগুলোর জন্য `.text-gradient` ক্লাস ব্যবহার করুন।
```tsx
<span className="text-gradient">কোর ইন্টেলিজেন্স</span>
```

## 📊 ৩. ডেটা ভিজ্যুয়ালাইজেশন

### রিচার্চস কনফিগারেশন
- **গ্রেডিয়েন্টস**: এরিয়া চার্টের জন্য `linearGradient` সবসময় ব্যবহার করুন (অপাসিটি 0.3 -> 0)।
- **টুলটিপস**: হাই-কনট্রাস্ট ডার্ক স্টাইল ব্যবহার করুন।
```tsx
contentStyle={{ 
  backgroundColor: 'rgba(13, 13, 18, 0.95)', 
  border: '1px solid var(--neon-blue)', 
  borderRadius: 8, 
  backdropFilter: 'blur(10px)' 
}}
```

### টেবিলস
- স্বাচ্ছন্দ্য ব্যাকগ্রাউন্ড ও নিয়ন হোভার স্টেটের জন্য `.cyber-table-row` ব্যবহার করুন।
- ডাটা কলামগুলোতে সংখ্যা এবং টাইমস্ট্যাম্পের জন্য `JetBrains Mono` ব্যবহার করুন।

## 🎬 ৪. মোশন ও ফিডব্যাক (`framer-motion`)

### পেজ ট্রানজিশন

