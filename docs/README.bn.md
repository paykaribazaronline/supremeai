# সুপ্রিম AI স্টুডিও 🚀
![Build Status](https://github.com/nazifarabbu/supremeai/workflows/SupremeAI%20Unified%20Pipeline/badge.svg)

**এআই-চালিত অ্যাপ ডেভেলপমেন্টের পরবর্তী প্রজন্ম**

সুপ্রিম AI স্টুডিও একটি এন্টারপ্রাইজ-গ্রেড মাল্টি-এজেন্ট প্ল্যাটফর্ম, যা স্বয়ংক্রিয় Android অ্যাপ জেনারেশন, সুরক্ষিত কোড বিশ্লেষণ এবং স্বয়ংক্রিয় ক্লাউড অর্কেস্ট্রেশনের জন্য ডিজাইন করা হয়েছে।

## 🌟 মূল বৈশিষ্ট্যসমূহ
- **প্রজেক্ট জেনারেটর**: স্বাভাবিক ভাষার রিকোয়ারমেন্টস ব্যবহার করে সুপ্রিম AI-চালিত এজেন্টদের মাধ্যমে সম্পূর্ণ অ্যাপগুলো তৈরি করা।
- **কোডফ্লো মডিউল**: 95% সুরক্ষা সনাক্তকরণ এবং 100-পয়েন্ট স্বাস্থ্য স্কোরিংয়ার সাথে রিয়েল-টাইম কোড বিশ্লেষণ।
- **এন্টারপ্রাইজ ই-কোলাবোরেশন**: উপস্থিতি ট্র্যাকিংয়ার সাথে রিয়েল-টাইম মাল্টি-ইউজার স্টুডিও এবং AI-চালিত জোট প্রোগ্রামিং।
- **ক্রস-প্ল্যাটফর্ম ইকোসিস্টেম**: VS Code, IntelliJ এবং মোবাইল (Flutter) এর জন্য এককীকৃত অভিজ্ঞতা।
- **ক্লাউড ন্যাটিভ**: GKE, Secret Manager এবং Artifact Registry-এর সম্পূর্ণ ইন্টিগ্রেটেড।

## 🚀 কিভাবে শুরু করবেন
১. **ইনফ্রাস্ট্রাকচার**: GKE সেটআপের জন্য [GOOGLE_ACQUISITION_PLAN.md](GOOGLE_ACQUISITION_PLAN.md) দেখুন।
২. **JDK 21**: SDKMAN-এর মাধ্যমে ইনস্টল করুন: `curl -s "https://get.sdkman.io" | bash && source "$HOME/.sdkman/bin/sdkman-init.sh" && sdk install java 21.0.5-tem`
৩. **ব্যাকএন্ড**: `./gradlew bootRun` (Java 21 & Virtual Threads প্রয়োজন)।
৩. **স্টুডিও**: `cd dashboard && npm run dev` (http://localhost:5173-এ পাওয়া যাবে)।
৪. **সিক্রেটস**: Secret Manager-এ `SUPREMEAI_API_KEY` সেট করা নিশ্চিত করুন।
