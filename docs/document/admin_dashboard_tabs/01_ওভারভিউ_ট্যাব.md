# ওভারভিউ ট্যাব (Overview Tab) - সুপ্রিমএআই অ্যাডমিন ড্যাশবোর্ড

## সংক্ষিপ্ত বিবরণ
ওভারভিউ ট্যাব হল সুপ্রিমএআইর মিশন কমান্ড সেন্টার। এটি একটি প্রিমিয়াম স্ট্যাটাস মনিটরিং ইন্টারফেস যা রিয়েল-টাইমে সিস্টেমের স্বাস্থ্য, কর্মক্ষমতা এবং নিরাপত্তা অবস্থা প্রদর্শন করে।

## মূল কম্পোনেন্ট

### ১. TelemetryBar
- **প্রপস**: `{ uptime, latency, sessions, apiStatus[] }`
- **কাজ**: ৫ সেকেন্ড পর পর আপডেট (পোলিং/WebSocket)
- **স্টেট**: সবুজ (ঠিক আছে), হলুদ (ডিগ্রেডেড), লাল (ক্রিটিক্যাল)

### ২. GlassKPICard
```tsx
interface GlassKPICardProps {
  title: string;          // কার্ডের শিরোনাম
  value: string | number; // মূল মান
  trend?: 'up' | 'down' | 'stable'; // ট্রেন্ড অভিযোজন
  unit?: string;          // একক (যেমন: %, ms)
  onClick?: () => void;   // ক্লিক হ্যান্ডলার
}
```
- **অ্যানিমেশন**: অ্যানিমেটেড নম্বর ট্রানজিশন, হোভার গ্লো
- **ক্লিক**: ঐতিহাসিক ভিউ মোডাল খোলে

### ৩. NeuralTerminal
- **ভার্চুয়ালাইজেশন**: react-window (৫০০+ এন্ট্রি)
- **অ্যানিমেশন**: ক্রিটিক্যাল লগের জন্য গ্লিচ এন্ট্রি
- **ফিল্টার**: INFO/WARN/CRIT টগল
