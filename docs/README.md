# SupremeAI Documentation Directory

এই ডিরেক্টরিতে SupremeAI প্রজেক্টের সমস্ত ডকুমেন্টেশন সুসংগঠিতভাবে রাখা হয়েছে।

## ফোল্ডার স্ট্রাকচার

- [**architecture/**](./architecture/): সিস্টেম ডিজাইন, আর্কিটেকচার এবং কুইক রেফারেন্স গাইড।
- [**deployment/**](./deployment/): GCP, Firebase এবং Cloud Run ডিপ্লয়মেন্ট গাইড।
- [**guides/**](./guides/): কন্ট্রিবিউটিং গাইড, কোড অফ কন্ডাক্ট এবং এরর-সলিউশন লগ।
- [**reports/**](./reports/): কোড কোয়ালিটি, কম্প্লিশন এবং ডুপ্লিকেট অ্যানালাইসিস রিপোর্ট।
- [**status/**](./status/): প্রজেক্টের বর্তমান স্ট্যাটাস, TODO লিস্ট এবং ইমপ্লিমেন্টেশন প্ল্যান।
- [**summaries/**](./summaries/): ফিক্স সামারি, ইম্প্রুভমেন্ট এবং মডিউল ইন্টিগ্রেশন রিপোর্ট।
- [**technical/**](./technical/): ডিবাগ অ্যানালাইসিস, লগইন ইস্যু ডায়াগনস্টিকস এবং টেকনিক্যাল মডিউল রিডমি।

## মূল ফাইলসমূহ
- [**AGENTS.md**](../AGENTS.md): এজেন্ট গাইডলাইন এবং রুলবুক (রুট ডিরেক্টরিতে)।
- [**README.md**](../README.md): প্রজেক্ট ওভারভিউ এবং কুইক স্টার্ট (রুট ডিরেক্টরিতে)।
