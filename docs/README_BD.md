# SupremeAI প্রকল্প গাইড

## প্রকল্প ওভারভিউ

SupremeAI একটি এআই-চালিত অ্যাসিস্টেন্ট প্ল্যাটফর্ম যা বিভিন্ন ক্ষেত্রে মেশিন লার্নিং এবং এআই টেকনোলজির সমন্বয় ধারণ করে। এই প্রকল্পটি একটি সম্পূর্ণ মডুলার আর্কিটেকচার দিয়ার সাথে তৈরি হয়েছে যা সহজেই ব্যবহারযোগ্য এবং রক্ষণশীল।

## প্রকল্প স্ট্রাকচার

```
supremeai/
├── config/          # কনফিগারেশন ফাইলগুলো
├── dashboard/       # রিয়াক্ট ড্যাশবোর্ড
├── docs/            # ডকুমেন্টেশন
├── functions/       # Firebase ফায়ারবেস ফাংশনগুলো
├── infra/           # ইন্ফ্রাস্ট্রাকচার সংক্রান্ত ফাইল
├── scripts/         # ইউটিলিটি স্ক্রিপ্টগুলো
├── src/             # মূল সোর্স কোড
├── supremeai/       # ফ্লাটার মোবাইল অ্যাপ
├── supremeai-studio-client/  # স্টুডিও ক্লায়েন্ট
└── supremeai-vscode-extension/ # VS Code এক্সটেনশন
```
