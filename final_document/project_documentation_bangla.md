# সুপ্রিমএআই প্রজেক্টের সম্পূর্ণ ডকুমেন্টেশন

## প্রজেক্ট ওভারভিউ

সুপ্রিমএআই স্টুডিও একটি এন্টারপ্রাইজ-গ্রেড মাল্টি-এজেন্ট প্ল্যাটফর্ম যা স্বয়ংক্রিয় অ্যান্ড্রয়েড অ্যাপ জেনারেশন, সিকিউর কোড অ্যানালাইসিস এবং স্বয়ংক্রিয় ক্লাউড অর্কেস্ট্রেশনের জন্য ডিজাইন করা হয়েছে। এটি এআই-চালিত অ্যাপ ডেভেলপমেন্টের পরবর্তী প্রজন্মের প্রতিনিধিত্ব করে।

### মূল বৈশিষ্ট্যসমূহ
- **প্রজেক্ট জেনারেটর**: স্বাভাবিক ভাষা থেকে সম্পূর্ণ অ্যাপ তৈরি করার জন্য সুপ্রিমএআই-চালিত এআই এজেন্ট
- **কোডফ্লো মডিউল**: রিয়েল-টাইম কোড অ্যানালাইসিস সহ ৯৫% সিকিউরিটি ডিটেকশন এবং ১০০-পয়েন্ট হেলথ স্কোরিং
- **এন্টারপ্রাইজ কোলাবরেশন**: রিয়েল-টাইম মাল্টি-ইউজার স্টুডিও প্রেজেন্স ট্র্যাকিং এবং এআই-চালিত পেয়ার প্রোগ্রামিং
- **ক্রস-প্ল্যাটফর্ম ইকোসিস্টেম**: ভিএস কোড, ইন্টেলিজ এবং মোবাইল (ফ্লাটার) জুড়ে ইউনিফাইড এক্সপেরিয়েন্স
- **ক্লাউড নেটিভ**: জিকেইই, সিক্রেট ম্যানেজার এবং আর্টিফ্যাক্ট রেজিস্ট্রি এর সাথে পুরোপুরি ইন্টিগ্রেটেড

## আর্কিটেকচার

### সিস্টেম ওভারভিউ
মনোরেপো যা স্বয়ংক্রিয় অ্যাপ জেনারেশনের জন্য মাল্টি-এজেন্ট এআই সিস্টেম রয়েছে।

### মডিউল স্ট্রাকচার
```
supremeai/
├── src/main/java/com/supremeai/    # স্প্রিং বুট ৩ ব্যাকএন্ড (জাভা ২১)
│   ├── codeflow/                    # কোডফ্লো অ্যানালাইসিস মডিউল (কোর ফিচার)
│   ├── ai/                         # মাল্টি-প্রোভাইডার এআই রাউটিং
│   ├── controller/                  # রেস্ট এপিআই এন্ডপয়েন্ট
│   ├── service/                     # বিজনেস লজিক
│   └── repository/                 # ফায়ারস্টোর ডেটা লেয়ার
├── dashboard/                       # রিয়েক্ট + টাইপস্ক্রিপ্ট + ভাইট ফ্রন্টএন্ড
├── supremeai/                       # ফ্লাটার অ্যাডমিন অ্যাপ (অ্যান্ড্রয়েড/আইওএস)
├── supremeai-vscode-extension/      # ভিএস কোড এক্সটেনশন (স্ক্যাফোল্ড)
├── supremeai-intellij-plugin/        # ইন্টেলিজ প্লাগইন (ভি১.২.০, কে২ রেডি)
├── command-hub/                      # সিএলআই টুলস
└── functions/                        # ফায়ারবেস ক্লাউড ফাংশনস
```

### কী ডিজাইন প্যাটার্নস
- **৩-লেয়ার ফ্লো**: কন্ট্রোলার (ভ্যালিডেশন) → সার্ভিস (বিজনেস লজিক) → রিপোজিটরি (ফায়ারস্টোর)
- **সিকিউরিটি**: সার্ভিস লেয়ারে পারমিশন চেক, জেডব্লিউটি অথ
- **ক্লাউড-ফার্স্ট**: ফায়ারবেস ব্যাকএন্ড, কোর ফিচারের জন্য কোনো লোকাল ফলব্যাক নেই
- **সোলো-ক্যাপেবল**: ফিচারগুলো অফলাইনে কাজ করে এআই প্রোভাইডার অনুপলব্ধ থাকলেও মিনিংফুল আউটপুট দেয়

### ক্রিটিকাল কম্পোনেন্টস
১. **কোডফ্লো মডিউল**: একমাত্র প্রোডাকশন-রেডি ফিচার। ৪০৯ জাভা ফাইল, ৫৩কে+ লাইন। ১৩+ ভালনারেবিলিটি স্ক্যানার সাপোর্ট করে।
২. **এআই প্রোভাইডার রাউটিং**: ১১ কানেক্টর সহ অটোমেটিক ফেইলওভার। এপিআই কীগুলো এনভায়রনমেন্ট ভেরিয়েবল থেকে।
৩. **ফায়ারবেস ইন্টিগ্রেশন**: অথ, ফায়ারস্টোর, ফাংশনস। ড্যাশবোর্ডে মডুলার এসডিকে, লেগ্যাসি পেজে কমপ্যাট।

## মডিউল বিস্তারিত

### ১. ব্যাকএন্ড (স্প্রিং বুট ৩)
- **টেকনোলজি**: জাভা ২১, স্প্রিং বুট ৩, গ্রেডল, ভার্চুয়াল থ্রেডস
- **কী ফিচারস**: রেস্ট এপিআই, এআই রাউটিং, কোডফ্লো অ্যানালাইসিস, ফায়ারবেস ইন্টিগ্রেশন
- **রান কমান্ড**: `./gradlew bootRun`
- **বিল্ড**: `./gradlew clean build -x test`

### ২. ড্যাশবোর্ড (রিয়েক্ট/টাইপস্ক্রিপ্ট)
- **টেকনোলজি**: রিয়েক্ট ১৮, টাইপস্ক্রিপ্ট, ভাইট, থ্রি.জেএস, অ্যান্ট ডিজাইন
- **কী ফিচারস**: ৩ডি রিয়েল-টাইম ভিজুয়ালাইজেশন, চার্ট, এআই মডেল ম্যানেজমেন্ট
- **রান কমান্ড**: `cd dashboard && npm run dev`
- **বিল্ড**: `npm run build`

### ৩. মোবাইল অ্যাপ (ফ্লাটার)
- **টেকনোলজি**: ডার্ট, ফ্লাটার, গুগল সাইন-ইন, ফায়ারবেস
- **কী ফিচারস**: অ্যাডমিন প্যানেল, রিয়েল-টাইম ডেটা, বাংলা লোকালাইজেশন
- **প্ল্যাটফর্ম**: অ্যান্ড্রয়েড/আইওএস

### ৪. ভিএস কোড এক্সটেনশন
- **টেকনোলজি**: টাইপস্ক্রিপ্ট, ভিএস কোড এপিআই
- **কী ফিচারস**: রিয়েল-টাইম কোড অ্যানালাইসিস, এআই সাজেশন, কোডফ্লো ড্যাশবোর্ড
- **কমান্ডস**: কোড অ্যানালাইসিস, এরর রেজোলিউশন, সিকিউরিটি ইস্যুস

### ৫. ইন্টেলিজ প্লাগইন
- **টেকনোলজি**: কোটলিন, ইন্টেলিজ প্ল্যাটফর্ম এপিআই
- **কী ফিচারস**: অ্যান্ড্রয়েড স্টুডিও ইন্টিগ্রেশন, কে২ মোড কমপ্যাটিবিলিটি
- **ভার্সন**: ১.২.০

### ৬. কমান্ড-হাব (সিএলআই)
- **টেকনোলজি**: নোড.জেএস, কমান্ড-লাইন টুলস
- **কী ফিচারস**: প্রজেক্ট ম্যানেজমেন্ট, ডিপ্লোয়মেন্ট স্ক্রিপ্টস

### ৭. সার্ভারলেস (ফায়ারবেস ফাংশনস)
- **টেকনোলজি**: নোড.জেএস, ফায়ারবেস ফাংশনস
- **কী ফিচারস**: ক্লাউড ফাংশনস, এপিআই প্রক্সি

## এআই মডেল রেজিস্ট্রি

- **জিপিটি-৪ও-মিনি**: স্ট্রাকচার্ড ডেটা এবং লজিক ভেরিফিকেশন ব্যাকআপ
- **ডিপসিক-ভি৪প্রো**: প্রফেশনাল কোডিং এবং টেকনিক্যাল আর্কিটেক্ট

### ফলব্যাক এবং ডেভেলপমেন্ট স্যুট (হাগিংফেস সার্ভারলেস)
- **কোডল্লামা-৩৪বি**: কোড জেনারেশনের জন্য প্রাইমারি ফলব্যাক

- **ল্লামা-৩-৮বি**: বাংলা ভাষা এবং কনভার্সেশনাল নিউয়ান্সের জন্য স্পেশালাইজড
- **ফাই-৩-ভিশন**: লাইটওয়েট ইমেজ অ্যানালাইসিসের জন্য স্পেশালাইজড এন্ডপয়েন্ট
- **মাল্টিলিঙ্গুয়াল-ই৫**: র্যাগ টাস্কের জন্য হাই-পারফরম্যান্স এমবেডিংস

### ইমারজেন্সি এবং ফাস্ট রেসপন্স (রেন্ডার ফ্রি টায়ার)

- **টিনিল্লামা-১.১বি**: বেসিক কানেক্টিভিটি টেস্টের জন্য অলওয়েজ-অন ইমারজেন্সি ফলব্যাক

## বাংলা ভাষা সমর্থন

### আন্তর্জালিকরণ (ইন্টারন্যাশনালাইজেশন)
- **ফাইলগুলো**: `dashboard/src/i18n/bn.json`, `supremeai/assets/i18n/bn.json`
- **ভিএস কোড এক্সটেনশন**: `supremeai-vscode-extension/package.nls.bn.json`
- **ভাষা কীগুলো**: `en` (ইংরেজি), `bn` (বাংলা)

### বাংলা যোগ করার নিয়মাবলী
১. **ইউজার ইন্টারফেস**: যেকোনো ইউজার-ফেসিং টেক্স্টের জন্য `bn.json` এ বাংলা যোগ করুন
২. **এরর মেসেজ**: সব এরর মেসেজের বাংলা অনুবাদ যোগ করুন
৩. **ডকুমেন্টেশন**: নয়, শুধুমাত্র ইংরেজিতেই রাখুন
৪. **কমেন্ট**: বাংলায় কমেন্ট করা যায় (বিশেশ করে জাভা/স্প্রিং বুট ফাইলে)

### ভাষা সনাক্তকরণ
- ব্যবহারকারীর ব্রাউজার/সিস্টেম ভাষা অনুযায়ী স্বয়ংক্রিয়ভাবে বাংলা দেখানো হবে
- `UserLanguagePreferenceService` ভাষা পছন্দ সংরক্ষণ করে
- এপিআই রেসপন্সেও ভাষা অনুযায়ী বাংলা উপযোগী হবে

- **ক্লাউডবিল্ড**: সিআই/সিডি পাইপলাইন
- **কুবারনেটিস**: অর্কেস্ট্রেশন
- **ডকার**: কন্টেইনারাইজেশন

- `feat:`: নতুন ফিচার
- `fix:`: বাগ ফিক্স
- `docs:`: ডকুমেন্টেশন
- `Cleanup:`: কোড ক্লিনআপ

- রিভিউয়ের জন্য স্ট্যান্ডার্ড ফরম্যাট মেইনটেইন করুন

## নিরাপত্তা এবং সেরা প্র্যাকটিস

### সিকিউরিটি
- সার্ভিস লেয়ারে সব পারমিশন চেক রাখুন
- কন্ট্রোলারে নয়
- এপিআই কীগুলো এনভায়রনমেন্ট ভেরিয়েবল থেকে রিড করুন
- হার্ডকোডেড সিক্রেটস এড়িয়ে চলুন

### কোড কোয়ালিটি
- স্প্রিং বুট ৩-লেয়ার ফ্লো মেইনটেইন করুন
- ফিচার প্যারিটি নিশ্চিত করুন: রিয়েক্ট ড্যাশবোর্ডে যা আছে তা ফ্লাটার অ্যাপেও থাকবে
- ক্লাউড-ফার্স্ট অ্যাপ্রোচ: লোকাল ফলব্যাকের উপর ক্লাউড প্রোভাইডার প্রায়োরিটাইজ করুন

## জিরো-হার্ডকোড অর্কেস্ট্রেশন পলিসি

- **ডাইনামিক ডিসকভারি**: সব এআই মডেল `AIProviderDiscoveryService` থেকে ফেচ করা হয়
- **অ্যাডমিন কন্ট্রোল**: ড্যাশবোর্ডে অ্যাডমিনদের প্রোভাইডার এনেবল/ডিসেবল, রোল অ্যাসাইন (কমিউনিকেশন, এক্সিকিউশন, ভোটিং) এবং এপিআই এন্ডপয়েন্ট আপডেট করার ক্ষমতা থাকবে
- **ফলব্যাক লজিক**: যদি কোনো হাই-প্রায়োরিটি মডেল ফেইল করে, সিস্টেম অটোমেটিক্যালি ডাইনামিক কনফিগ বেসড নেক্সট অ্যাভেইলেবল প্রোভাইডারে রোটেট করে
