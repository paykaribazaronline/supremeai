## ১. ওসিআর ট্যাব (OCR Tab)

### মূল কম্পোনেন্ট
`dashboard/src/components/AdminOCRCard.tsx`

### ভিজ্যুয়াল কম্পোনেন্টস
- **OCR Telemetry Strip**: ৩টি স্ট্যাটাস কার্ড (Neural Accuracy, Queue Depth, Total Scans)
- **Neural OCR Interface**: ড্র্যাগ অ্যান্ড ড্রপ আপলোডার
- **Extraction Manifest**: প্রক্রিয়াকৃত ফাইলের তালিকা

### ডেটা ইন্টারফেস
```typescript
interface OCRResult {
  id: string;
  fileName: string;
  text: string;
  confidence: number;  // 0.94-0.99
  status: 'processing' | 'completed' | 'error';
  byteSize?: string;
}
```

### কার্যকারিতা
- ফাইল আপলোড → ২ সেকেন্ড সিমুলেশন → প্রকাশ্য টেক্সট
- রিয়েল এপিআই নেই, স্থানীয় প্রক্রিয়াধীন

### ভিজ্যাল কম্পোনেন্টস
- **Threat Landscape KPIs**: ৪টি স্ট্যাটাস কার্ড (Critical Vectors, Active Scanners, Defense Integrity, Threat Index)
- **Vulnerability & Exploitation Matrix**: টেকনিকের টেবিল

### ডেটা ইন্টারফেস
```typescript
interface Technique {
  id: number;
  name: string;
  description: string;
  severity: 'CRITICAL' | 'HIGH' | 'MEDIUM' | 'LOW';
  status: string;
  detectionProbability?: number; // 50-90%
}
```

### API এন্ডপয়েন্ট
| মেথড | এন্ডপয়েন্ট |
|------|------------|
| GET | `/api/exploitation-techniques` |

### Severity রঙ
| স্তর | রঙ |
|------|-----|
| CRITICAL | লাল (red) |
| HIGH | নারঙ্গা (orange) |
| MEDIUM | নীল (blue) |
| LOW | সবুজ (emerald) |

## ৩. ভিপিএন ট্যাব (VPN Tab)

### মূল কম্পোনেন্ট
`dashboard/src/components/VPNManagement.tsx`

### ভিজ্যুয়াল কম্পোনেন্টস
- Alert Banner: VPN Management Description
- VPN Connections Table: তালিকা ও কানেক্ট/ডিসকানেক্ট অ্যাকশন
- Add/Edit Modal: VPN কনফিগারেশন ফর্ম

### ডেটা ইন্টারফেস
```typescript
interface VPNConnection {
  id: string;
  name: string;
  protocol: string;  // OpenVPN, WireGuard, L2TP, PPTP
  server: string;
  port: number;
  status: 'connected' | 'disconnected' | 'error';
  encryption: string;  // AES-256, AES-128, ChaCha20
  autoConnect: boolean;
  lastConnected: string;
}
```

### API এন্ডপয়েন্ট
| মেথড | এন্ডপয়েন্ট | বর্ণনা |
|------|------------|--------|
| GET | `/api/vpn/list` | সকল VPN তালিকা |
| POST | `/api/vpn/add` | নতুন VPN যোগ |
| PUT | `/api/vpn/{id}` | VPN আপডেট |
| POST | `/api/vpn/{id}/connect` | VPN কানেক্ট |
| POST | `/api/vpn/{id}/disconnect` | VPN ডিসকানেক্ট |
| DELETE | `/api/vpn/{id}` | VPN মুছে ফেলা |
