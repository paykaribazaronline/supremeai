### ডেটা ইন্টারফেস
```typescript
interface Requirement {
  id: string;           // proposalId
  title: string;        // প্রস্তাবের শিরোনাম
  description: string;    // বিস্তারিত বর্ণনা
  status: 'pending' | 'approved' | 'rejected';
  priority: 'low' | 'medium' | 'high' | 'urgent';
  source: string;         // ANALYZER, KNOWLEDGE_BASE, ইত্যাদি
  createdAt: string;
  impact?: number;        // ৩০-৯০% র‍্যান্ডম প্রভাব স্কোর
}
```

## ২. কীভাবে কাজ করে (Functionality)

### ২.১ মূল ফিচার

#### প্রস্তাব আনুগতি
- ১৫ সেকেন্ডে একবার `/api/admin/improvements/pending` এ ডেটা রিফ্রেশ
- `ImprovementProposal` অবজেক্টগুলোকে `Requirement` ইন্টারফেসে ম্যাপিং

#### প্রস্তাব অনুমোদন/প্রত্যাখ্যান
- **Approve**: `POST /api/admin/improvements/approve/{id}` → AdminDashboardService.approveProposal()
- **Reject**: `POST /api/admin/improvements/reject/{id}` → AdminDashboardService.rejectProposal()
- সফল হলে তালিকা থেকে সরিয়ে ফেলা হয়

### ২.২ API এন্ডপয়েন্ট

| মেথড | এন্ডপয়েন্ট | বর্ণনা |
|------|------------|--------|
| GET | `/api/admin/improvements/pending` | সকল পেন্ডিং প্রস্তাব |
| POST | `/api/admin/improvements/approve/{proposalId}` | প্রস্তাব অনুমোদন |
| POST | `/api/admin/improvements/reject/{proposalId}` | প্রস্তাব প্রত্যাখ্যান |

@GetMapping("/improvements/pending")
public Mono<ResponseEntity<ApiResponse<Map<String, Object>>>> getPendingImprovements() {

### ৩.৩ সার্ভিস লেয়ার (AdminDashboardService.java)

#### Auto-Pilot মোড
```java
public boolean submitImprovement(ImprovementProposal proposal) {
    if (isAutoPilotEnabled) {
        // স্বয়ংক্রিয়ভাবে অনুমোদন এবং নোটিফিকেশন যোগ
        proposal.approve();
        autoPilotNotifications.add("Auto-Pilot Action: " + proposal.getTitle());
        return true;  // শিখন প্রয়োগের অনুমতি
    } else {
        // অ্যাডমিন অনুমোদনের জন্য অপেক্ষায় রাখা
        pendingApprovals.put(proposal.getProposalId(), proposal);
        return false; // প্রয়োগের জন্য অপেক্ষা করুন
    }
}
```

#### ম্যানুয়াল অনুমোদন
```java
public boolean approveProposal(String proposalId) {
    ImprovementProposal proposal = pendingApprovals.remove(proposalId);
    if (proposal != null) {
        proposal.approve();
        log.info("Admin APPROVED: {}", proposal.getTitle());
        return true;
        // এক্সটেনশন: ইভেন্ট ট্রিগার করে মডিউলকে জানাতে পারেন
    }
    return false;
}
```

## ৪. প্রয়োজনীয়তা বিভাজন

| ক্যাটেগরি | উৎস | বর্ণনা |
|-----------|------|--------|
| KNOWLEDGE_BASE | GlobalKnowledgeBase | জ্ঞান আপডেটের প্রস্তাব |
| IMMUNITY_SYSTEM | ImmunitySystem | নিরাপত্তা উন্নয়নের প্রস্তাব |
| AI_PROFILER | Profiler | AI প্রোফাইল অপ্টিমাইজেশন |

## ৫. Priority লেভেল

| প্রায়োরিটি | রঙ | অর্থ |
|------------|-----|-----|
| urgent | লাল (red) | অতিব্যাপী গুরুত্বপূর্ণ |
| high | নারঙ্গা (orange) | উচ্চ অগ্রাধিকার |
| medium | নীল (blue) | সাধারণ অগ্রাধিকার |
| low | সবুজ (emerald) | কম গুরুত্বপূর্ণ |
