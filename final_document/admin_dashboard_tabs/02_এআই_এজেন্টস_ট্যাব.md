## ব্যাকএন্ড কন্ট্রোলার

### AIAgentsController.java
```java
@RestController
@RequestMapping("/api/ai-agents")
public class AIAgentsController {
    
    @GetMapping
    public Mono<ApiResponse<List<Agent>>> getAllAgents() {
        return agentRepository.findAll().collectList().map(ApiResponse::ok);
    }
    
    @GetMapping("/stats")
    public Mono<ApiResponse<Map<String, Object>>> getAgentStats() {
        return agentRepository.findAll().collectList().map(agents -> {
            Map<String, Object> stats = new HashMap<>();
            stats.put("totalAgents", agents.size());
            stats.put("activeAgents", agents.stream().filter(a -> "ACTIVE".equals(a.getStatus())).count());
            stats.put("idleAgents", agents.stream().filter(a -> "IDLE".equals(a.getStatus())).count());
            return ApiResponse.ok(stats);
        });
    }
    
    @PutMapping("/{id}/status")
    public Mono<ApiResponse<Agent>> updateAgentStatus(
        @PathVariable String id, 
        @RequestParam String status) {
        return agentRepository.findById(id)
            .flatMap(agent -> {
                agent.setStatus(status);
                return agentRepository.save(agent);
            })
            .map(ApiResponse::ok);
    }
}
```

## ডেটা মডেল

### Agent এন্টিটি
```java
@Entity
public class Agent {
    @Id
    private String id;
    private String name;
    private String type;  // llm, image, voice
    private String status; // ACTIVE, IDLE, ERROR
    private String uptime;
    private Integer load;  // 0-100%
    private Integer tasks;
}
```

## কাজের ধাপ

১. **ডেটা লোড**: কম্পোনেন্ট মাউন্ট হলে `fetchAgents()` কল
২. **স্ট্যাটাস রিফ্রেশ**: ৫ সেকেন্ড পর পর আপডেট
৩. **স্ট্যাটাস পরিবর্তন**: ব্যবহারকারী বাটন ক্লিক → `updateStatus()` API কল
৪. **লোড ভিজ্যুয়ালাইজেশন**: Progress কম্পোনেন্ট দিয়ে লোড প্রকাশ

## স্টেট ম্যানেজমেন্ট

| স্টেট | রঙ | অর্থ |
|------|------|------|
| ACTIVE | সবুজ (#10b981) | এজেন্ট কাজ করছে |
| IDLE | নীল (#3b82f6) | এজেন্ট প্রস্তুত আছে |
| ERROR | লাল (#ef4444) | এজেন্টে সমস্যা |

## নিরাপত্তা বিবেচনা

- শুধুমাত্র ADMIN ভূমিকা এজেন্ট স্ট্যাটাস পরিবর্তন করতে পারে
- API কলের সময় `authUtils.fetchWithAuth()` ব্যবহার করে টোকেন যাচাই
- ত্রুটি ঘটলে `message.error('LINK_COMMAND_FAILED')` দেখানো হয়
