# deploy/ সাবফোল্ডার গাইড

## উদ্দেশ্য

এই ফোল্ডারটি ডিপ্লোয়মেন্ট এবং প্রোডাকশন ম্যানেজমেন্ট সম্পর্কিত স্ক্রিপ্টগুলো ধারণ করে।

## ফাইলগুলো

| ফাইল                          | ব্যবহার                     |
| ----------------------------- | --------------------------- |
| `deploy.sh`                   | মূল ডিপ্লোয়মেন্ট স্ক্রিপ্ট |
| `get-fresh-token.js.example`  | টোকেন রিফ্রেশ উদাহরণ        |
| `setup-admin-user.js.example` | এডমিন ইউজার সেটআপ উদাহরণ    |

## ব্যবহার

```bash
# ডিপ্লোয়মেন্ট চালান
./deploy/deploy.sh
```
