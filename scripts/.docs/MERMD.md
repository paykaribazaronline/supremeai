# MERMD - Scripts

## Overview
Utility scripts for deployment, testing, and maintenance.

## How It Works

### Architecture Flow
```
Command → Script → System/API → Result
```

## Script Categories

### Auth Scripts (`scripts/auth/`)
- Authentication setup and management

### Deploy Scripts (`scripts/deploy/`)
- Deployment automation
- Environment setup

### Knowledge Scripts (`scripts/knowledge/`)
- Knowledge base seeding
- Data migration

### Test Scripts (`scripts/test/`)
- Automated testing
- Integration tests

### Tool Scripts (`scripts/tools/`)
- Development utilities
- Seed data generation (`scripts/tools/seed_data/`)

## Usage
- Run with Node.js or shell
- Environment variables for configuration
- Prerequisites documented per script
