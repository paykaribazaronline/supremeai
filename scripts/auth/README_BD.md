# auth/ সাবফোল্ডার গাইড

## উদ্দেশ্য

এই ফোল্ডারটি প্রমাণীকরণ এবং ইউজার ম্যানেজমেন্ট সম্পর্কিত স্ক্রিপ্টগুলো ধারণ করে।

## ফাইলগুলো

| ফাইল | ব্যবহার |
|------|--------|
| `add_admin.js` | নতুন এডমিন ইউজার যোগ করে |
| `debug_login.js` | লগইন সমস্যা ডিবাগ করে |
| `debug_login.js` | লগইন ডিবাগিং স্ক্রিপ্ট |

## ব্যবহার

```bash
# এডমিন ইউজার যোগ
node auth/add_admin.js --email=admin@example.com
```
