# seed/ সাবফোল্ডার গাইড

## উদ্দেশ্য

এই ফোল্ডারটি AI জ্ঞান ডাটাবেসকে পূর্ণ করার (সিড করার) স্ক্রিপ্টগুলো ধারণ করে।

## ফাইলগুলো

| ফাইল                                  | ব্যবহার                |
| ------------------------------------- | ---------------------- |
| `seed_part1_ai_fundamentals.py`       | এআই মৌলিক বিষয়        |
| `seed_part2_software_architecture.py` | মোতায়েন আর্কিটেকচার   |
| `seed_part3_databases.py`             | ডাটাবেস                |
| `seed_part4_security.py`              | সিকিউরিটি              |
| `seed_part5_devops_cloud.py`          | ডিভঅপস ও ক্লাউড        |
| `seed_part6_performance.py`           | পারফরম্যান্স           |
| `seed_part7_testing.py`               | টেস্টিং                |
| `seed_part8_system_design.py`         | সিস্টেম ডিজাইন         |
| `seed_part9_gemini_expert_wisdom.py`  | জেমিনি এক্সপার্ট উইজডম |
| `seed_resilience_knowledge.py`        | রেসিলিয়েন্স জ্ঞান     |
| `seed_copilot_knowledge.py`           | কোপিলট জ্ঞান           |

## ব্যবহার

# জ্ঞান সিড করন
