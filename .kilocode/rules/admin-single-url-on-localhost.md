# admin-single-url-on-localhost.md

Rule description here...Ensure the admin panel is accessible via a single URL on localhost with all features included.

## Guidelines

- The admin panel must be accessible via a single URL on localhost (e.g., http://localhost:3000/admin).
- All admin features must be available under this single URL.
- Avoid multiple URLs for different admin sections; consolidate under one base URL.
- Ensure the URL structure is consistent with deployed environments for easy migration.
