# SupremeAI VS Code Extension

**সুপ্রিমএআই - এআই-চালিত ডেভেলপমেন্ট সহায়ক**

একটি রিয়েল-টাইম মেশিন লার্নিং-ভিত্তিক VS Code এক্সটেনশন যা আপনার কোডিং প্যাটার্নকে শিখে এবং স্মার্ট সাজেশন দেয়।

## নতুন সংযোজিত মূল ফিচারসমূহ (New Features)

### ১. লগইন বাইপাস ও ফ্রি মডেল ফলব্যাক (Login Bypass & Fallback Routing)

- ব্যাকএন্ড সার্ভার ডাউন থাকলে বা লগইন না করা থাকলেও কোনো বাধা ছাড়াই কাজ করা যাবে।
- লোকাল **Ollama** (`http://localhost:11434/api/chat`) অথবা **OpenRouter Free API** মডেলে স্বয়ংক্রিয় ফলব্যাক রাউটিং সুবিধা।

### ২. অ্যাডমিন ও ইউজার ড্যাশবোর্ড (Admin & Customer Dashboards)

- এক্সটেনশন সাইডবারে যুক্ত হয়েছে নতুন দুটি প্যানেল:
  - **Admin Dashboard**: এআই প্রোভাইডার স্ট্যাটাস, মডেল ও অটো-লার্ন ট্র্যাকিং এবং সরাসরি কোডফ্লো এনালাইসিস কমান্ড ও চ্যাট অডিট ট্রিগার করার সুবিধা।
  - **User Dashboard**: ইউজারের প্রোফাইল, এডিট মেট্রিকস এবং কোড সাজেশনের ফিডব্যাক ডেটা ট্র্যাক করার প্যানেল।

### ৩. নিরাপদ টোকেন স্টোরেজ (SecretStorage)

- ওপেনরাউটার API কী এবং গিটহাব টোকেন এখন সাধারণ VS Code সেটিংসে প্লেইন টেক্সট হিসেবে না থেকে ওএসের সিকিউর স্টোরেজে (`SecretStorage`) নিরাপদভাবে সংরক্ষিত থাকে।

### ৪. রাইট-ক্লিক কনটেক্সট মেনু ও কমান্ড প্যালেট (Context Menu & Command Palette)

- এডিটরের যেকোনো কোড সিলেক্ট করে রাইট-ক্লিক মেনু অথবা কমান্ড প্যালেট থেকে সরাসরি **"SupremeAI: Send Selected Code to Chat"** কমান্ড দিয়ে কোডটি চ্যাট প্যানেলে পাঠানো সম্ভব।

## কার্যক্রম

### ১. রিয়েল-টাইম লার্নিং

- আপনার কোড এডিটকে স্বয়ংক্রিয়ভাবে শিখে
- ব্যক্তিগত কোড স্টাইল মেনে রাখে
- প্রজেক্ট-স্পেসিফিক সাজেশন দেয়

### ২. AI we সহায়তা

- **কোড পূরণ**: `Ctrl+Space` চাপুন অথবা কমান্ড প্যালেট থেকে `SupremeAI: AI Code Completion`
- **কোড ব্যাখ্যা**: সিলেক্ট করে `SupremeAI: Explain Code`
- **কোড রিভিউ**: `SupremeAI: AI Code Review`
- **চ্যাটে কোড পাঠানো**: সিলেক্ট করে `SupremeAI: Send Selected Code to Chat`

### ৩. CodeFlow বিশ্লেষণ

- ডিপেন্ডেন্সি গ্রাফ জেনারেট করে
- সিকিউরিটি ইস্যু ডিটেক্ট করে
- কোড হেলথ স্কোর গণনা করে

## কমান্ডস (বাংলা টাইটেলসহ)

| কমান্ড                                  | বাংলা টাইটেল                           |
| --------------------------------------- | -------------------------------------- |
| `SupremeAI: AI Code Completion`         | সুপ্রিমএআই: এআই কোড পূরণ               |
| `SupremeAI: Explain Code`               | সুপ্রিমএআই: কোড ব্যাখ্যা করুন          |
| `SupremeAI: AI Code Review`             | সুপ্রিমএআই: এআই কোড রিভিউ              |
| `SupremeAI: Send Selected Code to Chat` | সুপ্রিমএআই: সিলেক্টেড কোড চ্যাটে পাঠান |
| `SupremeAI: Force Learn`                | সুপ্রিমএআই: বর্তমান ফাইল থেকে শিখুন    |
| `SupremeAI: Run CodeFlow`               | সুপ্রিমএআই: কোড ফ্লো বিশ্লেষণ চালান    |

## ডেভেলপমেন্ট

```bash
# ইনস্টল ডিপেন্ডেন্সি
npm install

# কম্পাইল করুন
npm run compile

# ওয়াচ মোডে রান
npm run watch

# লিন্ট চেক
npm run lint

# টেস্ট রান
npm run unit
```
