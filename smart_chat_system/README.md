
# স্মার্ট চ্যাট সিস্টেম

স্মার্ট চ্যাট সিস্টেম একটি উন্নত চ্যাট প্ল্যাটফর্ম যা চ্যাট মেসেজ থেকে স্বয়ংক্রিয়ভাবে রুলস, প্ল্যান এবং কমান্ড ডিটেক্ট করতে পারে। এটি ইউজার বা এডমিনের সাথে কনফার্মেশনের মাধ্যমে নিশ্চিত হয় যে কোন মেসেজটি কোন ক্যাটাগরিতে পড়বে, এবং সেই অনুযায়ী ডাটাবেজে সেভ করে।

## প্রধান ফিচারসমূহ

1. **স্বয়ংক্রিয় ক্লাসিফিকেশন**: চ্যাট মেসেজ থেকে স্বয়ংক্রিয়ভাবে রুলস, প্ল্যান এবং কমান্ড ডিটেক্ট করে।

2. **কনফার্মেশন সিস্টেম**: ইউজার বা এডমিনের সাথে কনফার্মেশনের মাধ্যমে নিশ্চিত হয় যে কোন মেসেজটি কোন ক্যাটাগরিতে পড়বে।

3. **প্ল্যান অ্যানালিসিস**: নতুন প্ল্যানের সাথে বিদ্যমান প্ল্যানের সামঞ্জস্যতা বিশ্লেষণ করে এবং ভবিষ্যতের পূর্বাভাস প্রদান করে।

4. **ইমেজ আপলোড**: চ্যাটে ইমেজ সেন্ড করার সুবিধা প্রদান করে।

5. **বাংলা এবং ইংরেজি ভাষা সাপোর্ট**: বাংলা এবং ইংরেজি উভয় ভাষাতেই কাজ করতে পারে।

6. **এডমিন প্যানেল**: সমস্ত রুলস, প্ল্যান, কমান্ড এবং চ্যাট হিস্ট্রি ম্যানেজ করার জন্য একটি এডমিন প্যানেল প্রদান করে।

## ইনস্টলেশন

1. রিকোয়ারমেন্টস ইনস্টল করুন:

```bash
pip install -r requirements.txt
```

2. অ্যাপ্লিকেশন চালান:

```bash
python app.py
```

3. ব্রাউজারে ওপেন করুন:

```
http://localhost:5000
```

## ব্যবহার

### চ্যাট ইন্টারফেস

1. মেসেজ ইনপুট বক্সে আপনার মেসেজ লিখুন।
2. ইমেজ আপলোড করতে ইমেজ আইকনে ক্লিক করুন।
3. পাঠান বাটনে ক্লিক করে মেসেজ পাঠান।
4. সিস্টেম স্বয়ংক্রিয়ভাবে মেসেজ ক্লাসিফাই করবে এবং প্রয়োজনে কনফার্মেশন চাইবে।

### প্ল্যান অ্যানালিসিস

1. "প্ল্যান অ্যানালিসিস" ট্যাবে যান।
2. নতুন প্ল্যান টেক্সট এরিয়ায় আপনার প্ল্যান লিখুন।
3. "প্ল্যান বিশ্লেষণ করুন" বাটনে ক্লিক করুন।
4. সিস্টেম আপনাকে সামঞ্জস্যতা রিপোর্ট এবং ভবিষ্যত পূর্বাভাস দেখাবে।

### এডমিন প্যানেল

1. এডমিন প্যানেলে যেতে উপরের ডান কোণায় "এডমিন প্যানেল" বাটনে ক্লিক করুন।
2. ড্যাশবোর্ড, পেন্ডিং কনফার্মেশন, রুলস, প্ল্যান, কমান্ড এবং চ্যাট হিস্ট্রি ট্যাব থেকে নিজের প্রয়োজনীয় তথ্য দেখুন।
3. পেন্ডিং আইটেমগুলো কনফার্ম বা প্রত্যাখ্যান করুন।

## সিস্টেম আর্কিটেকচার

সিস্টেমটি নিম্নলিখিত মডিউলগুলো নিয়ে গঠিত:

1. **chat_classifier.py**: চ্যাট মেসেজ ক্লাসিফাই করার জন্য দায়ী।
2. **database_manager.py**: ডাটাবেজ অপারেশন পরিচালনা করে।
3. **smart_chat_system.py**: মূল সিস্টেম লজিক ধারণ করে।
4. **plan_analyzer.py**: প্ল্যান অ্যানালিসিস এবং ফিউচার স্টেট প্রেডিকশন করে।
5. **image_processor.py**: ইমেজ আপলোড এবং প্রসেসিং করে।
6. **app.py**: ফ্লাস্ক ওয়েব অ্যাপ্লিকেশন এবং API এন্ডপয়েন্ট।
7. **templates/**: HTML টেমপ্লেট ফাইল।
8. **static/js/**: জাভাস্ক্রিপ্ট ফাইল।

## API এন্ডপয়েন্ট

### চ্যাট রিলেটেড

- `POST /api/chat`: চ্যাট মেসেজ প্রসেস করে
- `GET /api/chat/history`: চ্যাট হিস্ট্রি পায়

### কনফার্মেশন রিলেটেড

- `POST /api/confirm`: আইটেম কনফার্ম বা প্রত্যাখ্যান করে
- `GET /api/pending`: পেন্ডিং কনফার্মেশনের তালিকা পায়

### ডাটা রিলেটেড

- `GET /api/rules`: সকল রুলসের তালিকা পায়
- `GET /api/plans`: সকল প্ল্যানের তালিকা পায়
- `GET /api/commands`: সকল কমান্ডের তালিকা পায়
- `GET /api/item`: আইডি দিয়ে একটি আইটেম পায়
- `GET /api/confirmations`: কনফার্মেশন হিস্ট্রি পায়

### প্ল্যান অ্যানালিসিস রিলেটেড

- `POST /api/plan/analyze`: নতুন প্ল্যানের সাথে বিদ্যমান প্ল্যানের সামঞ্জস্যতা বিশ্লেষণ করে

### ইমেজ রিলেটেড

- `POST /api/image/upload`: ইমেজ আপলোড করে
- `POST /api/image/upload-base64`: Base64 এনকোডেড ইমেজ আপলোড করে
