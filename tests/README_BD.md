## ফোল্ডার স্ট্রাকচার

```
tests/
├── package.json           # নদী প্যাকেজ
├── playwright.config.js   # প্লেলগ্রাউড কনফিগ
├── admin.spec.js         # এডমিন প্যানেল টেস্ট
├── api.spec.js           # API টেস্ট
├── firestore.spec.js     # ফায়ারস্টোর টেস্ট
└── README_BD.md          # এই ডকুমেন্ট
```

## রান করার উপায়

```bash
# টেস্ট রান করন
cd tests && npm test

# হেডেড মোডে (ব্রাউজার দেখতে)
cd tests && npm run test:headed
```

## টেস্ট ক্যাটাগরি
