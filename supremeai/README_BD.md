# supremeai/ ফোল্ডার গাইড

## উদ্দেশ্য

এই ফোল্ডারটি SupremeAI-র ফ্লাটার মোবাইল অ্যাপ্লিকেশন ধারণ করে। এটি iOS এবং অ্যান্ড্রয়েড প্ল্যাটফর্মের জন্য বিল্ড করা হয়।

## ফোল্ডার স্ট্রাকচার

```
supremeai/
├── lib/           # ফ্লাটার লাইব্রেরি
│   ├── main.dart  # অ্যাপ এন্ট্রি পয়েন্ট
│   ├── providers/ # স্টেট ম্যানেজমেন্ট
│   ├── screens/   # UI স্ক্রিনগুলো
│   └── widgets/   # রিয়ুসেবল উইজেট
├── android/       # অ্যান্ড্রয়েড প্ল্যাটফর্ম
├── ios/           # iOS প্ল্যাটফর্ম
├── web/           # ওয়েব বায়েস
└── test/          # ফ্লাটার টেস্ট
```

## প্রয়োজনীয় কমান্ড

```bash
# ডেভেলপমেন্ট
flutter run

# বিল্ড অ্যান্ড্রয়েড
flutter build apk

# বিল্ড iOS
flutter build ios

# টেস্ট রান
flutter test
```

## মূল ফাইলগুলো

- `lib/main.dart` - অ্যাপের মূল এন্ট্রি পয়েন্ট
- `pubspec.yaml` - ফ্লাটার প্যাকেজ কনফিগ
- `android/app/build.gradle` - অ্যান্ড্রয়েড বিল্ড কনফিগ

## প্যাকেজes

- `providers/` - BLoC/Provider ইমপ্লিমেন্টেশন
- `screens/` - ভিইউ স্ক্রিনগুলো
- `widgets/` - রিয়ুসেবল উইজেটস
