# সুপ্রিমএআই ব্যবহারকারী গাইড (বাংলা)

## 📄 পরিচিতি

সুপ্রিমএআই হলো একটি উন্নত মাল্টি-এজেন্ট সিস্টেম যা স্বয়ংক্রিয়ভাবে অ্যান্ড্রয়েড অ্যাপ তৈরি করে। এই গাইডে সিস্টেমটি কীভাবে ব্যবহার করবেন তা বর্ণনা করা হলো।

## 🚀 শুরুর পদক্ষেপ

### ১. অ্যাডমিন ড্যাশবোর্ডে লগইন করুন

1. ব্রাউজার খুলুন এবং এ ঠিকানাটি দিন: `http://localhost:8001/admin.html`
2. আপনার অ্যাডমিন ক্রেডেনশিয়াল দিয়ে লগইন করুন
3. ড্যাশবোর্ডে প্রবেশ করবেন

### ২. ভাষা পরিবর্তন করুন (ঐচ্ছিক)

ড্যাশবোর্ডের ডান উপরের কোণে ভাষা পরিবর্তনের অপশন আছে:

- **EN** - ইংরেজি
- **বাংলা** - বাংলা

আপনি যেকোনো ভাষা নির্বাচন করতে পারেন।

## 📱 অ্যাপ জেনারেট করা

### ধাপ ১: নতুন অ্যাপ তৈরি

1. ড্যাশবোর্ডে "New App" বোতামে ক্লিক করুন
2. নিচের তথ্যগুলো পূরণ করুন:
   - **App Name:** আপনার অ্যাপের নাম (উদাহরণ: MyFirstApp)
   - **Description:** অ্যাপটি সম্পর্কে সংক্ষিপ্ত বর্ণনা
   - **Category:** অ্যাপের ধরন (Business, Education, Health, ইত্যাদি)

### ধাপ ২: ফিচার নির্বাচন

আপনার অ্যাপে যে ফিচারগুলো থাকবে তা নির্বাচন করুন:

- ✅ User Authentication (ব্যবহারকারী প্রমাণীকরণ)
- ✅ Database (ডাটাবেস)
- ✅ Push Notifications (পুশ নোটিফিকেশন)
- ✅ Camera Access (ক্যামেরা অ্যাক্সেস)
- ✅ Location Services (লোকেশন সার্ভিস)

### ধাপ ৩: AI এজেন্ট ব্যবহার

সিস্টেমে অনেকগুলো এজেন্ট রয়েছে:

- **X-Builder:** কোড লিখে এবং বিল্ড করে
- **Z-Architect:** সিস্টেমের ডিজাইন করে
- **Code Reviewer:** কোড পর্যালোচনা করে

আপনি নির্বাচন করতে পারেন কোন এজেন্ট ব্যবহার করবেন।

### ধাপ ৪: জেনারেশন শুরু

"Generate" বোতামে ক্লিক করলে সিস্টেম স্বয়ংক্রিয়ভাবে কোড তৈরি করবে।

⏱️ এটি কয়েক মিনিট সময় লাগতে পারে।

## 🔄 অ্যাপের অগ্রগতি দেখা

জেনারেশন চলাকালীন আপনি অ্যাপের অগ্রগতি দেখতে পারেন:

- **Current Phase:** বর্তমান পর্যায়
- **Progress:** অগ্রগতির হার
- **Agent Status:** এজেন্টের অবস্থা

## 💾 অ্যাপ ডাউনলোড করা

জেনারেশন শেষ হলে:

1. "Download" বোতামে ক্লিক করুন
2. অ্যাপটি আপনার কম্পিউটারে সেভ হবে
3. আপনি এটি পরীক্ষা করতে পারেন

## ⚙️ সেটিংস ও কনফিগারেশন

### প্রোফাইল সেটিংস

1. প্রোফাইল আইকনে ক্লিক করুন
2. "Settings" নির্বাচন করুন
3. নিচের তথ্যগুলো আপডেট করতে পারেন:
   - Display Name (প্রদর্শন নাম)
   - Email (ইমেইল)
   - Password (পাসওয়ার্ড)
   - Language (ভাষা)

### Notification Settings

বিজ্ঞপ্তি সেটিংস থেকে আপনি নির্বাচন করতে পারেন:

- Email Notifications (ইমেইল বিজ্ঞপ্তি)
- Push Notifications (পুশ বিজ্ঞপ্তি)
- In-app Notifications (অ্যাপ ভিতরের বিজ্ঞপ্তি)
