# 🚀 SupremeAI - Multi-Agent App Generator System v3.1

## 🎯 Vision
"AI works, admin watches, approves when needed, and gets the APK."
Fully automated app development system where multiple AI agents collaborate to build, test, and deploy applications.

## 🏗️ Core Architecture
- **Layer 0: AI Brain** - Shared memory & performance scoreboard.
- **Layer 1: Cloud Brain (Firebase)** - Orchestration & Consensus engine.
- **Layer 2: AI Agents** - X-Builder, Y-Reviewer, Z-Architect.
- **Layer 3: App Generator** - Template-based code generation.

## 🛠️ Setup Instructions
1. **Java:** Install JDK 17+.
2. **Firebase:** Place `service-account.json` in `src/main/resources/`.
3. **API Keys:** Update your `.env` file with Gemini, DeepSeek, and other keys.
4. **Build:** Run `./gradlew build` to sync dependencies.
5. **Run:** Run `./gradlew run` or use the provided `.bat` files.

## 👑 Admin (King Mode) Rules
- **King Overwrite:** Admin can override any AI decision at any time.
- **70% Consensus:** AI loop continues until 70% approval is met.
- **Non-Blocking:** System continues other work if a task is waiting for human action (🚨).
