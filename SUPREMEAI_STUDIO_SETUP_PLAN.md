# SupremeAI Studio: Google Acquisition Setup Plan

গুগল কর্তৃক SupremeAI অধিগ্রহণের পর, আমাদের মূল লক্ষ্য হলো একটি অত্যাধুনিক, ক্লাউড-নেটিভ এবং এআই-ফার্স্ট IDE তৈরি করা—যার নাম হবে **"SupremeAI Studio"**। 

প্রজেক্টের লিড হিসেবে, আমি এই ট্রানজিশন এবং নতুন IDE সেটআপের জন্য যে পদক্ষেপগুলো গ্রহণ করবো, তার একটি পূর্ণাঙ্গ রূপরেখা নিচে দেওয়া হলো:

## ১. আর্কিটেকচার এবং টেকনোলজি স্ট্যাক (Technology Stack)

নতুন IDE-টি হবে অত্যন্ত ফাস্ট, লাইটওয়েট এবং ইন্টিগ্রেটেড।
* **Core Framework:** Electron (অথবা Tauri) ব্যবহার করা হবে ডেস্কটপ ক্লায়েন্টের জন্য, যাতে উইন্ডোজ, ম্যাক এবং লিনাক্স সব জায়গায় নির্বিঘ্নে চলে।
* **Editor Engine:** Microsoft Monaco Editor (যেটি VS Code-এ ব্যবহৃত হয়)। এটি এডিটিংয়ের ক্ষেত্রে বেস্ট পারফরম্যান্স নিশ্চিত করবে।
* **Frontend:** React.js (TypeScript সহ) এবং Vite। UI হবে একদম প্রিমিয়াম, গ্লাস-মরফিজম (Glassmorphism) ডিজাইন এবং ডার্ক থিম যুক্ত।
* **Backend Integration:** Google Kubernetes Engine (GKE) Autopilot এবং Cloud Run.
* **AI Engine:** Google Gemini 2.0 Pro এবং Flash মডেল।

## ২. গুগল ক্লাউড এবং জেমিনাই ইন্টিগ্রেশন (Google Integration)

পুরোনো সিস্টেম থেকে আমরা গুগল ইকোসিস্টেমে পুরোপুরি শিফট করবো:
1. **Gemini API:** `GeminiProvider.java` ব্যাকএন্ডে ইমপ্লিমেন্ট করা হবে। কোড জেনারেশন, ফিক্স এবং এনালাইসিস সব Gemini করবে।
2. **Google Secret Manager:** API keys এবং সেনসিটিভ ডেটা (যেমন `service-account.json`) সিক্রেট ম্যানেজারে রাখা হবে, গিট রিপোজিটরিতে নয়।
3. **Firebase Auth:** ফায়ারবেস অথেন্টিকেশন ব্যবহার করে ইউজারদের লগইন করানো হবে, তবে এখন এটি সরাসরি Google Workspace-এর সাথে সিঙ্ক করা থাকবে।
4. **Cloud Storage:** ইউজারের প্রজেক্ট ফাইল এবং স্ন্যাপশট ক্লাউড স্টোরেজে ব্যাকআপ রাখা হবে।

## ৩. SupremeAI Studio-এর কোর ফিচারসমূহ

নতুন IDE-তে আমরা যে বিষয়গুলোর উপর সবচেয়ে বেশি ফোকাস করবো:

* **Real-time Code Analysis:** টাইপ করার সাথে সাথে CodeFlow Analyzer কোড স্ক্যান করে বাগ ধরবে। (ইতিমধ্যেই আমরা `CodeAnalyzerTest` ফিক্স করেছি এবং রেজেক্স ইম্প্রুভ করেছি)।
* **AI Pair Programmer:** সাইডবারে একটি এআই চ্যাটবট থাকবে। ইউজার কোড সিলেক্ট করে জেমিনাইকে প্রশ্ন করতে পারবে।
* **One-Click Deploy:** IDE থেকে সরাসরি Google Cloud Run বা Firebase-এ কোড ডেপ্লয় করার বিল্ট-ইন ফিচার।
* **Cloud Sync:** যেকোনো পিসিতে লগইন করলেই ইউজারের সেটিংস, থিম এবং প্লাগিন সিঙ্ক হয়ে যাবে।

## ৪. প্রথম ধাপে করণীয় (Immediate Setup Steps)

1. **ইনিশিয়াল রিপোজিটরি সেটআপ:**
   - একটি নতুন ডিরেক্টরি তৈরি করবো: `supremeai-studio-client`
   - Electron, React এবং Monaco Editor ইন্সটল করবো।
2. **UI/UX ডিজাইন:**
   - TailwindCSS বা Vanilla CSS দিয়ে একটি স্লিক, মডার্ন ডার্ক থিম তৈরি করবো।
3. **API রাউটিং আপডেট:**
   - পুরোনো `https://supremeai-lhlwyikwlq-uc.a.run.app` বাদ দিয়ে নতুন গুগল এন্ডপয়েন্ট `https://ide-api.supremeai.google.com` ব্যবহার করবো। (ইতিমধ্যেই আমি এক্সটেনশন এবং স্ক্রিপ্টগুলোতে এই পরিবর্তন করেছি)।
4. **টেস্টিং এবং কোয়ালিটি কন্ট্রোল:**
   - ব্যাকএন্ডের `CodeAnalyzerTest`-এর যে ফেইলিং টেস্টগুলো ছিলো, সেগুলো ফিক্স করা হয়েছে এবং কিছু আনসাপোর্টেড ফিচার (Regex Limitations) ডিসেবল করা হয়েছে যাতে CI/CD পাইপলাইন পাস করে।

## ৫. সিকিউরিটি এবং ডেটা প্রাইভেসি

গুগলের স্ট্যান্ডার্ড অনুযায়ী কোনো হার্ডকোডেড ক্রেডেনশিয়াল থাকবে না। `service-account.json` রিমুভ করা হয়েছে। এখন থেকে শুধুমাত্র এনভায়রনমেন্ট ভ্যারিয়েবল এবং Google Secret Manager ব্যবহৃত হবে।
