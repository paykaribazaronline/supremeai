# সুপ্রিম এআই (SupremeAI) স্ক্রিপ্ট এবং ওয়ার্কফ্লো গাইড

এই নথিতে প্রজেক্টের প্রধান স্ক্রিপ্ট ফাইলগুলো, সেগুলোর প্রয়োজনীয়তা এবং কীভাবে চালাতে হবে তার একটি তালিকা দেওয়া হলো।

| স্ক্রিপ্ট ফাইলের নাম | কেন ব্যবহার করবেন | কীভাবে চালাবেন |
| :--- | :--- | :--- |
| `seed_copilot_knowledge.py` | Firebase এ GitHub Copilot সংক্রান্ত জ্ঞান (প্যাটার্ন, এরর ডিটেকশন এবং ফিক্স) যুক্ত করার জন্য। | `python seed_copilot_knowledge.py` |
| `seed_lib.py` | Firestore কানেক্টিভিটি এবং ব্যাচ রাইটিং এর জন্য পাইথন ইউটিলিটি। | সিডিং স্ক্রিপ্টের অংশ হিসেবে ব্যবহৃত হয়। |
| `errors.py` / `databases.py` | এরর-ফিক্স পেয়ার এবং ডাটাবেস প্যাটার্নের এনসাইক্লোপিডিয়া। | সিডিং স্ক্রিপ্টে ডেটা হিসেবে ইম্পোর্ট করা হয়। |
| `seed_zencoder_knowledge.py` | AI ইঞ্জিনিয়ারিং, অটোনোমাস এজেন্ট প্ল্যানিং এবং ফুল-স্ট্যাক ইন্টিগ্রেশন সংক্রান্ত বিশেষ জ্ঞান Firebase-এ সিড করার জন্য। | `python seed_zencoder_knowledge.py` |
| `BOOTSTRAP_AND_TEST.ps1` | প্রজেক্ট বিল্ড করা, অ্যাপ্লিকেশন চালু করা, অ্যাডমিন ইউজার তৈরি করা এবং লার্নিং কুয়েরি টেস্ট করার একটি পূর্ণাঙ্গ অটোমেশন স্ক্রিপ্ট। | `.\BOOTSTRAP_AND_TEST.ps1` (PowerShell-এ) |
| `simple_learn_test.ps1` | সরাসরি বিল্ড করা JAR ফাইলটি চালিয়ে লার্নিং সিস্টেমের কার্যকারিতা দ্রুত পরীক্ষা করার জন্য। | `.\simple_learn_test.ps1` (PowerShell-এ) |
| `PIPELINE_FIX_SCRIPTS.sh` | পাইপলাইনের সমস্যা যেমন missing Gradle wrapper বা ESLint কনফিগ ঠিক করার জন্য। | `bash PIPELINE_FIX_SCRIPTS.sh` |
| `e2e-tests.yml` | GitHub Actions-এর মাধ্যমে Playwright ব্যবহার করে সম্পূর্ণ সিস্টেমের এন্ড-টু-এন্ড (E2E) টেস্টিং স্বয়ংক্রিয় করার জন্য। | GitHub-এ স্বয়ংক্রিয়ভাবে চলে অথবা 'Actions' ট্যাব থেকে ম্যানুয়ালি চালানো যায়। |
| `project_to_markdown.py` | পুরো প্রোজেক্টের কোড একটি মাত্র Markdown ফাইলে রূপান্তর করার জন্য (Kimi/LLM এ ব্যবহারের জন্য)। | `python project_to_markdown.py` |

### প্রয়োজনীয় নোট:

১. **এনভায়রনমেন্ট ভেরিয়েবল:** স্ক্রিপ্টগুলো চালানোর আগে নিশ্চিত করুন যে আপনার সিস্টেমে `JWT_SECRET`, `SUPREMEAI_ADMIN_PASSWORD` এবং `SUPREMEAI_FIREBASE_WEB_API_KEY` এর মতো এনভায়রনমেন্ট ভেরিয়েবলগুলো সঠিকভাবে সেট করা আছে।
২. **Firebase কানেক্টিভিটি:** জ্ঞান সিডিং স্ক্রিপ্টগুলোর জন্য Firebase প্রোজেক্টের ক্রেডেনশিয়াল ফাইল থাকা আবশ্যক।
৩. **পাওয়ারশেল পারমিশন:** উইন্ডোজে স্ক্রিপ্টগুলো চালানোর জন্য `Set-ExecutionPolicy RemoteSigned` কমান্ডের প্রয়োজন হতে পারে।
